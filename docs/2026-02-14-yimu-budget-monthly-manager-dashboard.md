# yimu 参考：月度预算管理（总预算面板 + 双线趋势 + 分类预算拖拽 + 结转复制）

## 背景与目标
参考 yimu `BudgetManagerActivity` 的核心体验，在 Jive 中补齐一套“月度预算管理”主页面，并对齐 yimu 的总预算统计口径与结转复制逻辑。

本次交付覆盖你确认的 `1 + 2 + 3`：
1. 月度预算管理页面：总预算面板 + 预算趋势双线 + 分类预算列表（可拖拽排序）
2. yimu 总预算口径：`有效总预算 = max(手动总预算, 分类预算合计)`，并根据二者关系决定“总预算扣哪些分类”
3. 月初自动复制 + 结转（多余预算累加 / 超额支出扣除）

## 入口与页面结构
### 入口
`设置页底部菜单 → 预算管理` 现在进入 **月度预算管理**（新页面）。

页面右上角 “更多” 菜单：
- `全部预算`：打开旧的预算列表页（历史/自定义预算等）
- `预算排除`：进入不计入预算分类设置
- `预算设置`：进入预算偏好设置（包含结转与自动复制开关）

### 页面结构（BudgetManagerScreen）
- 顶部：月份选择（弹出底部月份列表）
- 总预算卡片：
  - 显示：有效总预算、剩余、支出、日均、日预算、今日剩余
  - 支持：编辑总预算金额
- 预算趋势：
  - 双线：累计支出（红）/ 剩余预算（绿）
  - 覆盖整月（未来日期为 0，累计线保持水平）
- 分类预算：
  - `添加分类预算`
  - 列表支持拖拽排序（`positionWeight` 持久化）

## 关键业务口径（对齐 yimu）
### 1) 有效总预算（effectiveAmount）
对总预算（`categoryKey == null`）：
- `手动总预算(含结转) = budget.amount + budget.carryoverAmount`
- `分类预算合计 = Σ(分类预算 amount+carryover)`, 并应用 “父分类覆盖子分类” 规则（见下）
- `有效总预算 = max(手动总预算(含结转), 分类预算合计)`

对分类预算（`categoryKey != null`）：
- `有效预算 = amount + carryoverAmount`

### 2) “父分类覆盖子分类” 的分类预算合计
当某个父分类存在分类预算时，其子分类预算被视为该父分类的“子分配”，**不再叠加进总合计**：
- `分类预算合计 = 所有分类预算之和 - (父分类已设置预算时的子分类预算之和)`

### 3) 总预算扣账口径（yimu 规则）
当且仅当：
- `手动总预算(含结转) <= 分类预算合计` 且 `分类预算合计 > 0`

则总预算 **只统计** “已设置预算的分类（父/子）” 的支出。

否则，总预算统计 **全部分类支出**（但仍排除设置为“不计入预算”的分类）。

说明：
- 不计入预算分两层：
  - 账单维度：`JiveTransaction.excludeFromBudget=true` 永远不计入预算
  - 分类维度：`JiveCategory.excludeFromBudget=true` 仅对 **总预算** 生效（分类预算不应用该过滤）

## 月初自动复制与结转（对齐 yimu）
### 复制触发（建议行为）
当进入“当月预算管理”页面，且开关开启时：
- 若当月**没有总预算**，且上月有总预算：复制一份到当月
- 若当月**没有任何分类预算**（含停用的也算存在），且上月有分类预算：复制上月所有分类预算到当月

### 结转规则
结转只在“复制生成当月预算”时生效，计算方式与 yimu 一致：
- 先计算上月该预算的：
  - `上月有效预算(含结转) = amount + carryoverAmount`
  - `上月已用 = usedAmount`（总预算按 yimu 扣账口径计算）
- 若开启 **多余预算累加** 且 `已用 < 上月有效预算`：
  - `本月 carryoverAmount = 上月有效预算 - 已用`（正数）
- 若开启 **超额支出扣除** 且 `已用 > 上月有效预算`：
  - `本月 carryoverAmount = 上月有效预算 - 已用`（负数）
  - 并限制 `amount + carryoverAmount >= 0`，最低扣至 0

### 相关设置项
`预算设置 → 周期与结转`：
- `每月自动复制预算`（默认开）
- `多余预算累加`（默认关）
- `超额支出扣除`（默认关）

## 排序与持久化
- 分类预算列表支持拖拽排序
- 持久化字段：`JiveBudget.positionWeight`（数值越大越靠前）

### 手动（手机）
1. 打开 `预算管理`，确认：
   - 顶部有“总预算”面板与“预算趋势”双线图
   - 下方分类预算可长按拖动排序
2. 设置两条分类预算，并把总预算设置为小于分类预算合计：
   - 再新增一笔“未设置预算的分类”的支出
   - 返回总预算面板，确认该笔支出不计入总预算使用
3. 开启 `多余预算累加/超额支出扣除` 后，确保上月有预算与支出数据：
   - 进入新月份的 `预算管理`
   - 若当月预算为空，确认自动复制并带上 `carryoverAmount`（在预算详情页可看到“结转调整”行）
