# 2026-02-13 分类图标风格 + 预算创建入口优化

本文记录并说明：

- 分类图标支持「默认彩色」+ 全局/系统混合策略 + 单分类强制单色
- 「分类图标风格」选择弹窗从页面底部弹出
- 预算管理页“创建预算”入口文案统一、右下角入口改为仅 `+` 图标
- 周期记账：选择分类支持图标显示、搜索，以及“点一级分类图标展开二级分类图标”

## 分类图标风格

### 目标

- 默认显示彩色分类图标（使用 `assets/category_icons/` 原始 PNG/SVG 颜色）
- 支持全局切换：
  - 全彩
  - 全部单色（跟随分类颜色）
  - 混合（系统分类单色/自定义分类彩色）
- 支持单个分类强制单色（无论全局选什么）

### 用户可见入口

- 设置页：`设置 -> 外观 -> 分类图标风格`
  - 点击某个选项会立即生效并自动关闭（无需再点“确定”）
  - `彩色（默认）`
  - `单色（全部跟随分类颜色）`
  - `混合（系统单色/自定义彩色）`
- 创建/编辑分类：开关 `图标强制单色（即使在彩色模式下也显示为单色，跟随分类颜色）`
- 分类管理列表：若该分类开启了强制单色，会展示 `单色` 徽标

### 行为说明

- **彩色（默认）**
  - 所有分类图标保持原始彩色；不会再对图标做“淡化/降饱和”处理
- **单色（全部跟随分类颜色）**
  - 所有分类图标转为单色，并跟随分类颜色
- **混合（系统单色/自定义彩色）**
  - 系统分类（`isSystem == true`）：单色 + 跟随分类颜色
  - 自定义分类（`isSystem == false`）：保持原始彩色
- **单分类强制单色**
  - 当某个分类开启 `图标强制单色`：该分类图标始终显示为单色并跟随分类颜色

### 技术实现（简述）

- 全局风格：`SharedPreferences`（key：`category_icon_style_v1`）
- 分类级覆盖：`JiveCategory.iconForceTinted`（Isar 持久化，并纳入备份/恢复）
- 图标渲染：`CategoryService.buildIcon()`
  - 当不需要 tint 时，直接返回原始 `Image/SvgPicture`，保持素材本色

### 验证方式

手工验证：

1. 打开 `设置 -> 外观 -> 分类图标风格`，切换三种模式，回到分类列表观察系统/自定义分类图标变化。
2. 打开任意分类编辑页，切换 `图标强制单色`，回到分类管理列表确认出现/消失 `单色` 徽标。

## 预算创建入口优化

### 目标

- 统一预算相关文案：全部用「创建预算」
- 预算管理页右下角入口更简洁：仅保留 `+` 图标（避免按钮过重、干扰列表）

### 当前行为

- 预算管理页右上角 `+`、空状态按钮、右下角 FAB 都会打开“创建预算”底部弹窗
- 右下角 FAB 改为 icon-only，并保留 `tooltip=创建预算`（长按可见）

### 验证方式

1. 进入 `预算管理`，确认右下角为 `+` 圆形按钮（无文字）。
2. 点击右上角 `+` / 空状态按钮 / 右下角 `+`，都能打开同一个创建预算弹窗。

## 周期记账：选择分类页优化

### 目标

- “新建周期规则”里选择分类时，能看到分类图标，并可快速选到二级分类。
- 一级分类较多时支持搜索。
- 选择后的分类回填到“新建周期规则”表单中，并显示图标。

### 当前行为

- 入口：`周期记账 -> 新建周期规则 -> 分类`
- 打开 `选择分类` 页面后：
  - 顶部可搜索（支持中文/拼音/首字母/图标名关键字）
  - 默认仅展示「自定义分类」（不包含系统分类库）
  - 默认展示一级分类列表
    - 点击 **一级分类图标**：展开/收起二级分类图标
    - 点击一级分类行：直接选择该一级分类
    - 展开后点击二级分类图标：选择该二级分类
  - 当输入搜索关键字时，展示平铺搜索结果（包含一级/二级分类）
- 分类图标渲染与“分类管理”一致：遵循 `设置 -> 外观 -> 分类图标风格`（默认彩色）。

### 验证方式

1. 进入 `周期记账 -> 新建周期规则`。
2. 点击 `分类`，进入 `选择分类` 页面。
3. 点击任意一级分类的图标，确认能展开看到二级分类图标；点二级分类后返回表单。
4. 确认表单的“分类”字段显示所选分类名称 + 图标。
